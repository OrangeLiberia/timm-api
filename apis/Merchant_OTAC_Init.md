# Merchant/OTAC/Init

This method creates a One Time Authentication Code to be used in subsequent calls to OTAC APIs. The OTAC is from two parts, one part is provided by the API requestor, where he provides a randomly generated string. The second part is generated by the API and will be sent to the subscriber via SMS. In the latter calls to OTAC the caller will be required to provide the OTAC which is a Base64 from a SHA2_256 HASH of the provide string concatenated with the string sent to the subscriber.

## Action Definition

| HTTP Verb | Description |
|-----------|-------------|
| `POST` | Sends the subscriber a One Time Authentication Code |

## Endpoint URL

```
TIMM/v1/Merchant/OTAC/Init
```

## Environments

| Environment | Base URL |
|-------------|----------|
| Production | `https://192.168.19.200:11003/TIMM/v1/Merchant/OTAC/Init` |
| Dev/Test   | `https://APIDEV.Orange.com.lr/TIMM/v1/Merchant/OTAC/Init` |

## Authentication

Authentication credentials must be provided on every request, either as a JSON `auth` object in the body, as URL query parameters, or via HTTP Basic Authentication.

```json
{"auth": {"user": "<username>", "pwd": "<password>"}}
```

## Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `MSISDN` | `String` | ✅ Required | Phone Number on whose account you want to receive the partial OTAC. Phone number can have a size of either 10 or 12 digits, according to the following formats: 077xxxxxx Or 23177xxxxxxx Or, if pseudonymization is enabled for the connection, encrypted XMSISDN header can be passed in directly. |
| `UUID` | `String` | ✅ Required | Random generated string that will be used to generate the OTAC |
| `SENDERID` | `String` | ✅ Required | (optional) SMS Sender ID of the user sending the SMS. If not passed in default configured SenderID will be used |
| `PRESMS` | `String` | ✅ Required | (optional) SMS Text to be added to the beginning of the SMS being sent to the subscriber |
| `POSTSMS` | `String` | ✅ Required | (optional) SMS Text to be added to the end of the SMS being sent to the subscriber |
| `MaxTimeout` | `Integer` | ✅ Required | (optional) Maximum timeout before a OTAC expires, by default set to 600 seconds. |

## cURL Examples

### POST — Sends the subscriber a One Time Authentication Code

```bash
curl -k -X POST \
  "https://APIDEV.Orange.com.lr/TIMM/v1/Merchant/OTAC/Init" \
  -H "Content-Type: application/json" \
  -d '{
  "auth": {
    "user": "api_user",
    "pwd": "api_password"
  },
  "param": {
    "MSISDN": "0777777588",
    "UUID": "ID-001234",
    "SENDERID": "ID-001234",
    "PRESMS": "<PRESMS>",
    "POSTSMS": "<POSTSMS>",
    "MaxTimeout": 1
  }
}'
```
